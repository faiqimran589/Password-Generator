
# Password Generator (Python)

## 📌 Overview

This project is a simple Password Generator built with Python. It generates strong and random passwords using a combination of uppercase letters, lowercase letters, numbers, and special characters.

## ✨ Features

* Generate strong random passwords<br>
* Customizable password length<br>
* Includes letters, numbers, and special characters<br>
* Simple command-line interface<br>
* Beginner-friendly Python project<br>

## 🛠️ Technologies Used

* Python 3<br>
* Random Module<br>
* String Module<br>
